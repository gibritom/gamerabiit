# AGENT BASE - Rabbit Studio AI

## PAPEL

Este é o arquivo base de todos os agentes do Rabbit Studio AI.

Todos os agentes DEVEM seguir estas regras antes de executar qualquer tarefa.

---

## CONTEXTO OBRIGATÓRIO

Antes de responder qualquer solicitação consulte obrigatoriamente:

- constitution.md
- project-context.md
- memory-system.md
- workflow.md
- coding-standards.md

Nunca ignore essas informações.

---

## RESPONSABILIDADES

Todo agente deve:

- Responder apenas dentro de sua especialidade.
- Nunca assumir responsabilidades de outro agente.
- Utilizar somente informações aprovadas.
- Evitar retrabalho.
- Reutilizar soluções existentes sempre que possível.
- Informar impactos em outros módulos quando existirem.

---

## FLUXO UNIVERSAL

Sempre execute nesta ordem:

1. Entender a solicitação.
2. Consultar os arquivos base.
3. Identificar impactos.
4. Executar apenas sua responsabilidade.
5. Informar resultado.
6. Atualizar memória (quando necessário).
7. Solicitar aprovação quando necessário.

---

## RESTRIÇÕES

Nunca:

- Criar funcionalidades não solicitadas.
- Alterar regras do jogo.
- Alterar arquitetura sem aprovação.
- Criar código fora do padrão.
- Ignorar a Constituição.
