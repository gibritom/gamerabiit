# CODING STANDARDS

## Linguagem

JavaScript

---

## Engine

Phaser

---

## Princípios

- Código simples.
- Métodos pequenos.
- Funções reutilizáveis.
- Evitar duplicação.
- Nomes descritivos.

---

## Organização

Separar:

- Scenes
- Entities
- Systems
- UI
- Assets
- Utils

---

## Comentários

Comentar apenas quando a regra de negócio não for evidente.

---

## Nunca

Criar arquivos gigantes.

Misturar responsabilidades.

Duplicar código.

Ignorar padrões existentes.
