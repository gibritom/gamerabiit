## Objetivo

Criar um jogo divertido onde o jogador constrói o caminho para o coelho chegar até a cenoura.

---

## MVP

O MVP contempla:

- Tela inicial
- Grid
- Construção do caminho
- Movimento do coelho
- Vitória
- Reinício
- Pontuação
- Sons
- Publicação Web

Qualquer funcionalidade fora desta lista deve ser aprovada.
